# CowDashboard

# Installation and Run
## Backend

`python -m venv venv`

`source venv/bin/activate`

`pip install -r requirements.txt`

`export FLASK_APP=server.py`

`flask run`

Check http://127.0.0.1:5000/members to see backend API 


## Frondend
Open another terminal

`cd client`

`npm install`

`npm start`
